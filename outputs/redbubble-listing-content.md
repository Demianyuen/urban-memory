## Redbubble SEO Tips

### Title Best Practices
- Lead with the most specific keyword: "Hong Kong Central Blueprint Map Art"
- Include secondary keywords: "Architectural City Poster | Urban Wall Decor"
- Max 50 characters for front-loaded keywords
- Include the city name, style, and product type

### Tag Best Practices
- Use ALL 50 allowed tags per design
- Mix broad (city map, wall art) and specific (hong kong central, blueprint)
- Include style-specific tags (architectural print, minimalist, vintage)
- Include gifting tags (housewarming gift, anniversary gift)
- Location-based tags (hong kong decor, hong kong gift)
- Format-specific tags (metal print, poster, sticker)

### Description Best Practices
- Front-load important keywords
- Describe the design's emotional appeal
- Explain the style concept
- Mention geographic accuracy/real data
- List available product formats at the bottom
