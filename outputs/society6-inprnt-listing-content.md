# PART 2: INPRNT

## Platform Overview

INPRNT is a curated print-on-demand platform with higher quality standards and a more art-focused audience than Redbubble. They review each design before approval. INPRNT artists earn 65% profit margin (highest in POD).

### How INPRNT Works

- Artist uploads design at 300 DPI, minimum 3000px longest side
- INPRNT reviews for quality and originality (24-48 hours)
- Once approved, design is available on multiple products
- Artist sets their own markup (base price + artist markup)
- 65% artist margin on art prints, lower on other products
- INPRNT handles all printing, shipping, customer service
- Payouts monthly via PayPal

# PART 3: ADDITIONAL PLATFORMS (Quick Notes)

## Amazon Handmade

- Application required (approval can take weeks)
- 15% referral fee (lower than regular Amazon)
- Professional account: $39.99/month
- Best for metal prints and framed art (Amazon's fulfillment is excellent)
- **Not recommended as first priority** — approval delay is too long

## Fine Art America / Pixels.com

- Same company as INPRNT, broader product range
- 50+ products available
- Less curated, easier to upload
- Use same designs as INPRNT
- Lower margins than INPRNT but higher traffic
- **Secondary priority** — upload after INPRNT is set up

## Creative Market

- Curated marketplace for design assets
- One-time fee per sale (not print-on-demand)
- Best for selling the digital SVG/DXF/DWG files directly
- $25/year seller fee + 30-40% commission
- **Good fit for the digital product tier** — users download files to use in their own projects

## Displate

- Specializes in metal prints only
- Artist earns royalty on each sale
- Limited to 2D artwork (no 3D/spatial designs)
- **Consider after Redbubble + Society6 are live**

# PART 5: IMAGE EXPORT INSTRUCTIONS

## Batch Export Script

To upload to these platforms, export the SVG posters as high-res PNGs:

```bash
# Using Inkscape command line (requires Inkscape installed)
# For each SVG file:

inkscape input.svg --export-filename=output.png --export-width=6000 --export-height=4000

# Or using the Python/svglib approach:
python -c "
import cairosvg
for i in range(1, 7):
    svg = f'outputs/posters/style_{i}.svg'
    png = f'outputs/exports/style_{i}.png'
    cairosvg.svg2png(url=svg, write_to=png, output_width=6000, output_height=4000)
    print(f'Exported {png}')
"
```

## Manual Export via Browser (Alternative)

1. Open each SVG in Chrome/Firefox
2. Right-click → Save as... → PNG
3. Set zoom to 200% before saving for higher resolution
4. Crop in any image editor to exact dimensions
